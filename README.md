## Installation

This package can be deployed automatically using NPM:

```
npm i @bu0nq/uikit-scss-reboot
 ```

## Usage

SCSS files located in the `root` directory:

* `reboot.scss`

### Importing

To import all fonts, you can use:

```scss
@use "~@bu0nq/uikit-scss-reboot/reboot";
```
